# Power

Power functions and operators for Vector and Matrix structures.

## Overview

Raise each element in a vector or matrix to the power of the given exponent. An array of exponents is also supported.

## Topics

### Vector power

- ``pow(_:_:)-3kbly``
- ``pow(_:_:)-6g69x``
- ``pow(_:_:)-3czzt``
- ``pow(_:_:)-4ox3u``
- ``^(_:_:)-1014e``
- ``^(_:_:)-1bj2u``

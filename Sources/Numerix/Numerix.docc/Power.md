# Power

Power functions for Vector and Matrix structures.

## Overview

Raise each element of in a vector or matrix to the power of the given exponent.

## Topics

### Vector power

- ``pow(_:_:)-3kbly``
- ``pow(_:_:)-6g69x``
- ``pow(_:_:)-3czzt``
- ``pow(_:_:)-4ox3u``

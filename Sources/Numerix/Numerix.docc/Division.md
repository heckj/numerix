# Division

Division operators for Matrix structures.

## Overview

Matrix division using double precision values. The operations are element-wise.

## Topics

### Scalar-Matrix division

- ``/(_:_:)-6lkbt``
- ``/(_:_:)-7owya``

### Matrix-Scalar division

- ``/(_:_:)-6xulo``
- ``/(_:_:)-9x0k``
- ``/=(_:_:)-10szp``
- ``/=(_:_:)-35sxh``
